#Без повторов
Нужно вывести на сайте список всех режиссеров, которые находятся в этой базе. 
Для удобства чтения режиссеры должны быть представлены без повторов.

Пример результата:

+----------------------+
|       director       |
+----------------------+
|  Jorge Michel Grau   |
|     Gilbert Chan     |
|     Shane Acker      |
+----------------------+

Напиши запрос и выведи режиссеров.
